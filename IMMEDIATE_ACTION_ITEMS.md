**Impact**: +2 points (judges notice confident vs. stumbling)

**6 Key Questions to Rehearse**:

1. **"Why RandomForest instead of Deep Learning?"** (target: 35-40 sec)
   ```
   Answer (outline):
   - RandomForest: interpretable (show feature importance)
   - Works with small data (3K samples, DNNs need 100K+)
   - Fast inference (45ms, real-time capable)
   - More trusted by emergency responders (explainable)
   - Our F1=0.9886 proves it's sufficient
   - With limited historical floods (3-4/year in Da Nang), 
     DNNs would overfit and fail on new patterns
   ```

2. **"What if your model makes a wrong prediction?"** (target: 30-35 sec)
   ```
   Answer (outline):
   - Three safeguards:
     1. Confidence threshold (only alert if confidence > 85%)
     2. Human-in-the-loop (operator reviews, approves before dispatch)
     3. Feedback loop (operator marks predictions correct/incorrect)
   - Quarterly retraining with new data
   - Override always possible
   - We can never be 100% certain, but human approval mitigates risk
   ```

3. **"How can you claim F1=0.9886 with only 3K samples?"** (target: 35-40 sec)
   ```
   Answer (outline):
   - Proper methodology: 5-fold cross-validation (F1=0.9886 ± 0.0038 = STABLE)
   - High-quality features (water level + rainfall directly predict floods)
   - Class weight balancing (handles imbalance)
   - Conservative estimate: we document honestly
   - On test data (600 holdout samples), achieved this F1
   - Will validate further post-deployment with real-world data
   ```

4. **"Can this work in Bangkok or Jakarta?"** (target: 40-45 sec)
   ```
   Answer (outline):
   - Yes, architecture is modular
   - Parameterized: city, sensor locations, thresholds, districts
   - Backend code is 95% reusable
   - Main challenge: getting historical flood data per city
   - Solution: Start with physics-based synthetic data (like we did)
   - Integrate real data once available
   - Timeline: 2-3 weeks to deploy in new city
   ```

5. **"How much real data vs synthetic data?"** (target: 30-35 sec)
   ```
   Answer (outline):
   - 2,000 real samples (Da Nang 2019-2024 VNMHA)
   - 1,000 synthetic (physics-based simulation)
   - Why? Da Nang has only 3-4 major floods/year = limited real data
   - Synthetic extends diversity, prevents overfitting
   - Model trained on mix achieves F1=0.9886 on holdout real data
   - Transparent: we document this, judges know methodology
   ```

6. **"What's your evidence for '50-100 lives saved per year'?"** (target: 40-45 sec)
   ```
   Answer (outline):
   - Based on medical research linking evacuation speed to survival
   - Baseline: 2-hour evacuation (manual coordination)
   - With AegisFlow: 15-minute evacuation (optimized route)
   - 8x faster = 3-7% increase in survival rate
   - Da Nang: ~1,500 people per flood × 3-4 floods/year
   - Math: 1,500 × 3-7% = 45-105 additional lives/year
   - Conservative: report 50-100 lives/year
   - Will validate post-deployment with real outcomes
   ```
